jButton4ActionPerformed(java.awt.event.ActionEvent evt)

El siguiente código Java muestra una nueva ventana de registro de base de datos al hacer clic en un botón.

    
        private void jButton4ActionPerformed(java.awt.event.ActionEvent evt) {
            new registroBBDD().setVisible(true);
            dispose();
        }

Este método crea una nueva instancia de la ventana de registro de base de datos (clase `registroBBDD`), la hace visible y luego cierra la ventana actual.
